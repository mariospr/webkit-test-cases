This tests that an animation with a non-zero animation-delay and animation-fill-mode = forwards will keep the value specified at the 100% keyframe, even if the animation was instantaneous.

The placeholder text below should NEVER dissapear (but it does!):

Lorem ipsum dolor sit amet, consectetur adipiscing elit.

The placeholder text below won't dissapear because of using a zero-value for animation-delay:

Lorem ipsum dolor sit amet, consectetur adipiscing elit.

The placeholder text below won't dissapear because of using animation-fill-mode = none:

Lorem ipsum dolor sit amet, consectetur adipiscing elit.

The placeholder text below won't dissapear because of a non-zero value for animation-duration:

Lorem ipsum dolor sit amet, consectetur adipiscing elit.
